Задание 2.4.1 (HW-01)
«Установка среды редактора кода VS Code в операционной системе Windows»

1. Заходим по ссылке.
https://code.visualstudio.com/

2. Клик по кнопке [Download for Windows].

3. В открывшемся окне усановки VS Code - клик по кнопке [Принять условия соглашения].

4. Продолжение установки: Далее \ Далее \ Установить \ Завершить. => VS Code установлен.
P.S.
Путь установки редактора не меняем и оставляем по умолчанию (выбор пути установки может быть не предоставлен вовсе).
По умолчанию программа устанавливается в директорию AppData - (C:\Users\User01\AppData).
